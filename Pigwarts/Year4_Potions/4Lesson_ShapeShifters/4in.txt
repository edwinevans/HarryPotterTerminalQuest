Dictionary

in works on String, List, Tuple, and Dictionary and tells you if the collection contains the given value.

For example:
>>> "mockingbird" in { "mockingbird" : "sky", "worm" : "ground" }
True

>>> "sky" in { "mockingbird" : "sky", "worm" : "ground" }
False
